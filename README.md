#  Portfolio – Christian IMBHA

A personal website built with **HTML** and **Tailwind CSS**, showcasing my academic background, professional experiences, skills, and projects.  
This portfolio is designed to be **modern, responsive, and minimalist**, highlighting my profile as a web developer.

---

## Features

- **Fixed header** with clickable logo and responsive burger menu for mobile.
- **Clear navigation**: CV, Projects, Contact.
- **Structured sections**:
  - Academic and professional background
  - Internships and experiences
  - Skills and expertise
  - Languages
- **Responsive design**: centered text on mobile, left-aligned on desktop.
- **Elegant color palette** adapted to realistic photo backgrounds for readability.

---

##  Technologies Used

- **HTML5** – site structure
- **Tailwind CSS** – modern and responsive design
- **GitHub** – version control and deployment
- **JavaScript** (lightweight) – burger menu functionality
